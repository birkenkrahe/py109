Review DataCamp lesson 2 - Procedural programming

Concepts

  • What's a procedure?

    A series of steps that can be referenced and reused.

  • How are procedures implemented in Python?

    As functions, e.g. def hello(): print("hello")

  • When is procedural programming used?

    When a problem can be broken down into steps.

Code

  • Define two variables my_height and your_height and assign values to them, e.g. 180 cm and 170 cm. 

    my_height = 180
your_height = 175

  • Make a list height_list that contains the heights of four people: 179 cm, 185 cm, 175 cm and 189 cm.

    Tip: [1,2,3] is a list of numbers. 

    height_list = [179,185,175,189]

  • Write a conditional statement (with if) that checks if my_height is greater than your_height. If it is, print I'm taller. 

    if my_height > your_height:
    print("I'm taller")

  • Write a loop (with for) that loops over the variable height in height_list and prints out the height for every element of height_list: 

    for heigth in height_list:
    print(height)

  • Put the conditional inside the loop and if my_height is greater than the height element of the list, then print I'm taller than and the height. 

    for height in height_list:
    if my_height > height:
        print("I'm taller", height)

Creativity

  • Create a function hello that says "Hi, [yourname]": 

    def hello():
    print("Hi, Marcus")

  • Call the function. 

    hello()

  • Modify the function and rename it to hello_name: add name as an argument, and print name. Call the function with different names. 

    def hello_name(name):
    print(name)
    return

hello_name("Marcus")
hello_name("Jane")
hello_name("John")

  • Add a conditional to the function and rename it to hello_if: print the name only if it begins with the letter 'J'. 

    def hello_if(name):
    if name[0]=='J':
        print(name)
    return

hello_if("Marcus")
hello_if("Jane")
hello_if("John")

  • Add a loop to the function and rename it to hello_for:

    1. Change the argument to name_list.
    2. Loop over name_list and greet each name with "hello".
    3. Outside the function, before calling it, define a list of names name_list with the names "Saige","Lou","Ethan","Loren".
    4. Call the function with the argument name_list.
    def hello_for(name_list):
    for name in name_list:
        print("Hello,", name)
    return

name_list = ["Saige","Lou","Ethan","Loren"]
hello_for(name_list)
